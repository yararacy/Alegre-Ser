# Alegre-Ser — Plataforma Digital

Site oficial do Alegre-Ser: expansão de consciência para pessoas, famílias e empresas.

## Sobre o projeto

Plataforma institucional do Alegre-Ser, desenvolvida com HTML/CSS puro.
Baseada no Manual de Marca Alegre-se (Althaea Studio, 2023-24) e na
Análise Completa de Posicionamento (2026).

## Estrutura de arquivos

```
alegre-ser-plataforma/
│
├── index.html          ← Página principal (em construção)
├── README.md           ← Este arquivo
└── assets/             ← Imagens, logos e fontes (a adicionar)
    ├── images/
    └── fonts/
```

## Identidade visual

**Cores principais (Manual de Marca):**
- Laranja terracota: `#E89566` — cor de destaque primária
- Creme/fundo: `#F2EDE3`
- Dourado olive: `#C1AF5F`
- Teal: `#91A19B`
- Violeta suave: `#D19CBF`

**Tipografia:**
- Primária (títulos): Noto Serif Display — Google Fonts
- Secundária (corpo): DM Sans — Google Fonts
